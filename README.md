# Claude Code Plugin Marketplace

A personal plugin marketplace for [Claude Code](https://claude.com/code).

## Quick Start

### Add this marketplace

```bash
/plugin marketplace add krmcbride https://github.com/krmcbride/claude-plugins
```

### Install a plugin

```bash
/plugin install essentials@krmcbride
```

### List available plugins

```bash
/plugin marketplace list krmcbride
```
